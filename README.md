## NPM Publish

It should respect https://semver.org/ versionning

```sh
npm version [<newversion> | major | minor | patch]
git push --follow-tags
```

## NPM Publish alpha version
```sh
npm version prerelease --preid=alpha
git push --follow-tags
```
